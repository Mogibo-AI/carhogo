# ペルソナ定義 — CarHogo

## ペルソナ 1: ドライバー（MVP 中心ユーザー）

### 文野 もじゃ（Bunno Moja）— 毎日通勤するサラリーマン

| 属性 | 詳細 |
|------|------|
| **年齢** | 35歳 |
| **職業** | 営業職（自動車通勤） |
| **運転経験** | 10年以上 |
| **デバイス** | Pixel Watch（Wear OS）・Android スマートフォン・ノート PC またはタブレット（ダッシュボード設置） |
| **主な運転シーン** | 毎朝の高速道路通勤（片道60〜90分）・週1〜2回の長距離営業移動 |

#### 目標
- 事故なく安全に目的地へ到着したい
- 渋滞・長距離運転による疲労やストレスを最小化したい
- 遅刻時の連絡を手間なく済ませたい（スマートフォンを操作したくない）

#### 課題・痛点
- 長距離運転中に眠気を自覚したときにはすでに反応速度が低下している
- 割り込みなどで怒りが爆発しそうになっても自分でコントロールするのは難しい
- 渋滞中に遅刻確定でもスマートフォン操作は危険だが、連絡しないのも気まずい

#### CarHogo との主なインタラクション
1. 出発前に Pixel Watch の「開始」ボタンを押してモニタリングを起動する
2. 運転中は何も操作しない — CarHogoが自動で状態を監視する
3. 眠気・怒りが検知されたとき、Nova 2 Sonic の声に応答して会話する
4. 遅刻確定時、Nova 2 Sonic が読み上げるメッセージに「送って」と応答する
5. 目的地到着後に Pixel Watch の「終了」ボタンを押してモニタリングを停止する

#### CarHogoに期待すること
> 「自分が頑張らなくていい。CarHogoが全部やってくれる。ハンドルを握っていればいい。」
